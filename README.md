## Athena0.1（研究型项目）

本项目将 **“持续成长的认知架构”** 作为研究对象，而不是把目标限定为训练更大的模型。

### 核心文档（项目主线）
- **推荐阅读入口（新）**：`AthenaBrain/docs/00_Vision.md`（包含 Athena 身份与研究姿态）
- `AthenaBrain/docs/02_Cognitive_Spec.md`：Phase 1 “Apple Test”验收标准（从零开始概念形成）
- `AthenaBrain/docs/03_Architecture.md`：模块与最小闭环（对应验收标准）
- `AthenaBrain/docs/08_State_Manager.md`：External/Internal State、状态迁移与内部驱动机制
- `AthenaBrain/docs/07_Research_Log.md`：研究日志（所有实验/修改必须记录）

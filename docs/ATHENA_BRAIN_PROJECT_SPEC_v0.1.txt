Project Goal
Athena Brain is NOT another Large Language Model.
Athena Brain is an experimental cognitive architecture.
The goal is to verify whether an artificial intelligence can gradually grow its cognition throughout its lifetime, instead of obtaining all knowledge through one-time pretraining.
LLM is only used as the language interface.
The real intelligence should continuously grow from experiences.

Design Philosophy
Current LLMs have several fundamental limitations.
* Knowledge is stored inside parameters.
* Models are mostly static after training.
* Long-term learning is difficult.
* Concepts are implicit.
* Personal experiences are difficult to preserve.
* New knowledge often requires fine-tuning or retraining.
Athena Brain attempts another path.
Instead of building a larger model, we attempt to build a growing brain.

Core Philosophy
The AI should be born almost empty.
It should not know what dogs, apples, trees or mathematics are.
Everything should gradually emerge from interaction.
Knowledge should originate from experiences.
Concepts should emerge from many experiences.
Reasoning should emerge from concepts.
Creativity should emerge from accumulated understanding.

Three Core Objects
Everything inside Athena Brain is built upon three objects.
Experience
Concept
Relation
Everything else is derived from them.

Experience
Experience represents one real interaction with the world.
Examples:
User uploads a dog image.
User says:
"This is a dog."
Athena asks questions.
User answers.
This entire interaction becomes one Experience.
Experience fields
* id
* timestamp
* source
* input type
* image
* text
* audio
* conversation
* Athena response
* user feedback
* confidence
* tags
* processed flag
Experience is immutable.
Experiences represent Athena's life.

Sleep Engine
Sleep is one of the core innovations.
Every night Athena performs:
Replay today's experiences
Remove duplicated information
Merge repeated observations
Detect conflicts
Increase confidence
Decrease confidence
Generate abstractions
Generate new concepts
Generate questions
Compress memory
Update Concept Graph
Sleep is NOT idle time.
Sleep is cognition.

Introduction
This document proposes one of the most important ideas inside Athena Brain.
Current LLMs are mostly parameter-based intelligence.
Knowledge is encoded into billions of parameters.
Learning means changing parameters.
Athena Brain follows another philosophy.
Intelligence is represented by continuously evolving cognitive states.
Parameters are only one implementation detail.
The brain itself is a continuously changing state machine.

Core Principle
Knowledge is not stored.
Knowledge is continuously reconstructed.
Every new experience changes the internal state.
The brain is never static.

State 0
Raw Observation
Raw Observation contains only sensor data.
Examples
Pixels
Audio waveform
Touch
Temperature
Motion
No semantic meaning exists here.
The brain has not yet interpreted anything.

State 1
Perception
Perception transforms raw observations into recognizable features.
Examples
Red
Round
Soft
Moving
Four legs
Tail
Still no meaning.
Perception only describes what is observed.

State 2
Attention
Attention determines what deserves cognition.
Thousands of observations occur every second.
Only a very small number become experiences.
Attention may depend on
Novelty
Conflict
Importance
Danger
Reward
Curiosity
Emotion (future)
Goal (future)
Without attention,
nothing enters memory.

State 3
Experience
An Experience is a meaningful event.
Example
Today
User pointed to an object.
User said
"This is an apple."
Athena asked
"What can apples do?"
The user answered
"They can be eaten."
This entire interaction becomes one Experience.
Experience never changes.
Experiences represent Athena's life history.

State 4
Working Memory
Working Memory contains
Today's conversations
Temporary hypotheses
Pending questions
Unverified observations
Working Memory is volatile.

State 5
Long-term Memory
Sleep Engine transfers selected experiences into long-term memory.
Long-term memory stores
Stable knowledge
Stable experiences
Frequently reused information

State 6

State 7
Mental Model
Mental Models describe how Athena believes the world works.
Examples
Dogs usually bark.
Objects fall downward.
Fire is dangerous.
People answer questions.
Mental Models are predictive.
They are not absolute truth.
They continuously evolve.

State 8
Theory
Theory is a collection of Mental Models.
Examples
Physics
Biology
Language
Mathematics
A Theory is Athena's highest level of abstraction.
Theories explain
why
instead of
what.

State 9
Planning
Planning predicts future actions.
Examples
If I ask another question,
I may improve my understanding.
If confidence is low,
I should collect more evidence.
Planning transforms cognition into behavior.

State 10
Action
Athena performs actions.
Ask questions.
Observe.
Speak.
Move.
Search.
Actions generate new observations.
The loop begins again.

Conflict Detection
Every state may conflict.
Examples
Apple is red.
Apple is green.
Both may be true.
The brain should preserve conflicts.
Conflicts trigger curiosity.
Curiosity generates new observations.
New observations resolve conflicts.

Curiosity
Curiosity is generated automatically.
Examples
Unknown relation.
Conflicting evidence.
Low confidence.
Incomplete concept.
Prediction failure.
Curiosity drives learning.

Forgetting
Athena should not remember everything forever.
Forgetting is necessary.
Possible forgetting
Unused memories
Low-confidence hypotheses
Duplicate observations
Temporary working memory
Concepts should almost never disappear.
Experiences should almost never be deleted.
Only attention changes.

Identity influences attention.
Attention influences learning.
Learning changes identity.
Identity evolves continuously.
